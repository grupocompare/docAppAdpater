# AppAdapter — Doc Site (GitHub Pages)

Arquivos:
- `index.html`
- `styles.css`
- `app.js`
- `data.json`

## Como publicar no GitHub Pages

1. Crie uma pasta `docs/` (ou use a branch `gh-pages`).
2. Copie estes arquivos para dentro de `docs/`.
3. Em **Settings → Pages**, selecione:
   - **Deploy from a branch**
   - Branch: `main`
   - Folder: `/docs`
4. Acesse a URL do Pages.

## Atualizar conteúdo
Edite `data.json` (ou gere novamente a partir do `DOCUMENTACAO-AppAdapter.md`).
